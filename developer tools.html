أدوات المطورين

مجموعة من الأدوات والبرامج التي تُستخدم في بيئة عمل المبرمجين.

برامج ومحررات الأكواد

Visual Studio Code

محرر أكواد مفتوح المصدر من مايكروسوفت، يدعم العديد من اللغات البرمجية.

Visual Studio

مجموعة من الأدوات والبرمجيات المفتوحة المصدر التي تُستخدم مع Visual Studio لتسهيل عملية تطوير التطبيقات.

Pycharm

بيئة تطوير متكاملة (IDE) للغة بايثون، توفر أدوات قوية لتسهيل عملية البرمجة.

VSCodium

حلول مفتوحة المصدر مصممة خصيصًا للعمل بكفاءة داخل VSCodium وتوفير تجربة تطوير خالية من التتبع.

Sublime Text

محرر نصوص قوي وسريع يدعم العديد من اللغات البرمجية ويوفر ميزات متقدمة.

Atom

محرر نصوص مفتوح المصدر من GitHub، يوفر واجهة مستخدم قابلة للتخصيص بالكامل.

Notepad++

محرر نصوص خفيف الوزن يدعم العديد من اللغات البرمجية ويتميز بالسرعة والكفاءة.

WebStorm

بيئة تطوير متكاملة (IDE) قوية مخصصة لتطوير تطبيقات الويب باستخدام JavaScript وTypeScript.

online compilers

onlinecompilerHTML&CSS&JS (ببساطه)

محرر أكواد مفتوح المصدر من قبل مطورين موقع ببساطه يدعم (HTML&CSS&JS).

compilers all languages (ببساطة)

محرر أكواد مفتوح المصدر من قبل مطورين موقع ببساطه يدعم (all languages).

Replit

بيئة تطوير متكاملة عبر الإنترنت تدعم أكثر من 50 لغة برمجية مع إمكانية مشاركة المشاريع والتعاون .

CodePen

منصة تطوير مشهورة للواجهة الأمامية تدعم HTML و CSS و JavaScript مع معاينة مباشرة للنتائج.

JSFiddle

محرر أكواد عبر الإنترنت يركز على تطوير الويب ويدعم JavaScript و CSS و HTML.

CodeSandbox

بيئة تطوير متكاملة عبر الإنترنت مخصصة لتطوير تطبيقات الويب مع دعم لإطارات العمل الحديثة.

OnlineGDB

محرر ومصحح أخطاء عبر الإنترنت يدعم C و C++ و Java و Python والعديد من اللغات الأخرى.

PlayCode

محرر JavaScript سريع عبر الإنترنت مع دعم للمكتبات الشائعة والمعاينة المباشرة.

برامج تصميم المواقع

Figma

محرر تصميم واجهات المستخدم عبر الإنترنت مفتوح المصدر.

XD

محرر تصميم واجهات المستخدم عبر الإنترنت مفتوح المصدر.
